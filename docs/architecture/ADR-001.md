# ADR-001: İçerik Odaklı Yerel Öncelikli Modüler Monolit

- Durum: Kabul edildi
- Tarih: 2026-07-17
- Mimari Sorumlu: Kenan AY

## Karar

Uygulama Astro ve TypeScript üzerinde statik üretilecek; kanonik içerik Markdown/JSON dosyalarında, kişisel kullanıcı verisi IndexedDB'de tutulacaktır. Alan çekirdeği depolama teknolojilerinden portlar aracılığıyla ayrılır.

## Sonuçlar

İlk sürüm backend veya mikroservis gerektirmez. Çevrimdışı kullanım ve Git tabanlı içerik sürümleme basitleşir. Çok kullanıcılı senkronizasyon gerekirse ayrı bir API adaptörü daha sonra eklenebilir.
